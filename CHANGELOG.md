# Changelog

## 2026-06-15

### Added

- 新增仓库级 `README.md`，系统整理项目背景、流程演进、使用方式和业务实践记录
- 新增 `LICENSE`，采用 `MIT` 开源协议
- 新增 `项目成果说明.md`，用于展示项目目标、工作流价值和阶段性成果

### Improved

- 支持从教师列表页直接识别教师链接并进入个人主页补采
- 支持根据链接与页面内容自动识别真实学校、学院并修正误分类
- 支持跨 HTML 标签邮箱提取
- 支持 `#` 代替 `@` 的邮箱归一化
- 支持浏览器风格请求头和 `Referer` 缓解部分高校站点 `403`
- 支持过滤学院公邮、页脚公邮，默认仅保留含个人邮箱的有效数据
- 支持清洗姓名中的前缀/后缀职称，如教授、副教授、讲师、高级工程师、高级实验师等
- 支持一行多个 URL 自动展开处理
- 支持部分 JS 动态列表页和 Visual SiteBuilder 组件页解析
- 支持 OCR 识别图片形式的邮箱或个人信息
- 支持 HTML 残留、UTF-8 文本残留和中英双版链接清理

### Workflow

- 本地仓库已同步推送到 GitHub 与 Gitee
- GitHub 与 Gitee 均已配置公开仓库和 SSH 推送

## Earlier
